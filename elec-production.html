World Electricity Generation

A multi-line graph illustrating the sources used to generate electric power

Project Description

This graph showcases the percentages of the world's total primary energy supply that is used to generate electric power.

The main outcome of this project was to showcase my ability in creating a multi-line graph that is responsive and gives tooltips for exactly x-axis values.

Project Technicals

  • Graph created using d3.js v5 library
  • Data collected from the World Bank Open Data
  • Data format is CSV
  • The Multi-line graph is responsive and will adjust to any screen size
  • Colours representing the sources used in the lines/legend have different hues representing different values, generated from Color Brewer 2.0
  • Tooltip showing percentages appears on hover
